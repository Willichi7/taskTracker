# Task Tracker

A simple command-line interface (CLI) application to track and manage tasks. This project allows users to create, update, delete, and list tasks, along with prioritization and search functionality.

## Features

- **Add Tasks**: Create new tasks with an optional priority.
- **Update Tasks**: Change the status of tasks.
- **Delete Tasks**: Remove tasks by ID.
- **List Tasks**: View all tasks or filter by status (e.g., todo, in-progress, done).
- **Task Prioritization**: Set task priority (high, medium, low).
- **Sort Tasks**: Sort tasks based on creation or update dates.
- **Search Tasks**: Find tasks by keywords in their descriptions.
- **Help Command**: Display a list of available commands.

## Requirements

- Node.js installed on your machine.
